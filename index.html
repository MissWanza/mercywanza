Introduction to CSS basics

CSS (Cascading Style Sheet) is the code that styles web content and like HTML, CSS is not a programming language. It's not a markup language either.

Get to know more about CSS

CSS is a style sheet language that selectively styles HTML elements. It describes how the HTML elements are displayed on screen, paper, or in other media.

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

The Anatomy of a CSS ruleset

The whole structure is called a ruleset often referred to as just rule. Here are the individual parts of the CSS Rule:

  1. Selector: This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled.
  2. Declaration:This is a single rule like color. It specifies which of the element's properties you want to style.
  3. Properties: These are ways in which you can style an HTML element. (In this example, color is a property of the elements.) In CSS, you choose which properties you want to affect in the rule.
  4. Property value: To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property.

To Note

Here are other important parts of the syntax:

  • Apart from the selector, each ruleset must be wrapped in curly braces. ({})
  • Within each declaration, you must use a colon (:) to separate the property from its value or values.
  • Within each ruleset, you must use a semicolon (;) to separate each declaration from the next one.
